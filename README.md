# Chip-8-Interpreter


<h3>
A basic Chip-8 Interpreter written in C++.<br>
It can run *.ch8 ROMs from the default filepath.<br>
Currently working on Windows, <b>SDL2</b> (Simple DirectMedia Layer).<br>
</h3>
